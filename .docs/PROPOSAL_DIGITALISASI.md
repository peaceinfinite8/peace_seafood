# Proposal Digitalisasi Operasional

Ringkasan penawaran untuk digitalisasi operasional gudang ikan dan cold storage.

## Masalah utama

- Pencatatan manual rawan salah input dan manipulasi.
- Informasi stok dan kapasitas tidak real-time.
- Piutang, retur, dan laporan keuangan terlambat terpantau.

## Solusi inti

- Input timbangan riil dari checker.
- Dashboard executive untuk bos.
- Notifikasi stok minimum dan piutang jatuh tempo.
- Export laporan dan ringkasan keuangan.

## Role ringkas

- `bos`: lihat laporan dan approve retur.
- `admin`: operasional, nota, keuangan, master data.
- `checker`: input stok dan timbangan.
- `super_admin`: konfigurasi, user, gudang, backup.

## Implementasi bertahap

1. Uji coba paralel.
2. Pelatihan lapangan.
3. Migrasi data master.

## Catatan

Dokumen ini bersifat proposal. Detail implementasi tetap mengikuti struktur kode dan PRD yang aktif.
